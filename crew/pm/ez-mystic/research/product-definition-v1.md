# ez-mystic 产品定义文档

> 版本: 1.0 | 日期: 2026-05-14 | PM: ez-mystic

## 1. 产品定位

**东方神秘学 AI 平台，面向海外用户**

- **核心价值**：用 DeepSeek AI 解读中国传统命理（八字、紫微斗数、风水），以 Apple 风格 UI 呈现
- **目标用户**：海外华人 + 东方文化爱好者 + 西方灵性消费者
- **差异化**：全功能（八字+紫微+风水）+ AI 深度解读 + 多语言 + 海外支付

## 2. MVP 范围（Phase 1）

### 核心功能（必须有）
1. **八字排盘 + AI 解读**
   - 输入：出生年月日时 + 地点（时区）
   - 输出：四柱八字、五行分析、十神格局、大运流年
   - AI 解读：性格特征、事业运势、财运趋势、感情走向
2. **个人运势报告（命书）** — 付费核心
   - 20+ 页 PDF 报告
   - 性格 + 事业 + 财运 + 感情综合解读
   - 定价：$29.99（参考 FateTell $39.99）

### 次要功能（Phase 2）
3. **紫微斗数排盘 + AI 解读**
4. **风水分析**（户型上传 + AI 分析）
5. **合婚匹配**
6. **AI 对话**（持续陪伴式咨询）

### 不支持（MVP 阶段）
- ❌ 奇门遁甲（复杂度太高）
- ❌ 多语言（先英文，后续加）
- ❌ iOS App（先 Web 验证）

## 3. 产品架构

```
用户输入（生日/地点）
    ↓
规则引擎（排盘）← 100% 准确，纯算法
    ↓
结构化数据（八字/紫微/风水）
    ↓
AI 解读层（DeepSeek API + qwen35b）← 个性化解读
    ↓
RAG 知识库（古籍 + 案例）← 深度和质量保障
    ↓
输出优化 + 多语言 ← Apple 风格 UI
    ↓
付费下载（命书 PDF / 订阅）
```

## 4. 定价策略

| 产品 | 价格 | 说明 |
|------|------|------|
| 单次八字解读 | $9.99 | 入门体验 |
| 命书 PDF | $29.99 | 核心付费产品 |
| 月度订阅 | $4.99/月 | 持续运势更新 |
| 年度订阅 | $39.99/年 | 省 40%，主推 |
| AI 顾问 | $14.99/月 | 高端，Phase 2 |

## 5. 技术选型

| 层 | 技术 | 理由 |
|----|------|------|
| Web | Next.js | 已有技术栈，SSR 利于 SEO |
| App | React Native | 复用 Next.js 逻辑 |
| 排盘引擎 | Python/JS 自研 | 规则算法，避免 AI 幻觉 |
| AI 推理 | DeepSeek API + qwen35b | DeepSeek 为主，qwen35b 备选 |
| 知识库 | Chroma/Pinecone | RAG 向量检索 |
| 支付 | Dodo Payments / Paddle → Payoneer | 中国收款，MoR 合规 |
| 部署 | Vercel | 快速上线 |

## 6. 合规要点

- 定位为「娱乐/教育用途」，加免责声明
- 避免使用 "fortune telling"、"predict" 等敏感词
- 使用 "Chinese astrology"、"Eastern wisdom"、"self-discovery"
- GDPR/CCPA 数据合规
- 年龄分级 16+
