# 🎨 ez-mystic UI 设计指南

> 版本: 1.0 | 日期: 2026-05-14 | 设计师: ez-mystic

---

## 1. 设计理念

**Apple 的克制 × 东方神秘学的深邃**

- **Apple 设计语言**：极简、留白、圆角、毛玻璃（Glassmorphism）、微动画
- **东方神秘学氛围**：暗色基调、金色点缀、星象纹理、水墨意境
- **核心原则**：
  - 内容优先，装饰克制
  - 每一次交互都有反馈（微动效、触感）
  - 留白是奢侈，不是浪费
  - 神秘感来自"少"，不是"多"

## 8. 纹理与装饰元素

### 8.1 星象纹理

- **类型**：极淡的星图 / 二十八宿线稿
- **位置**：首页 Hero 背景、结果页装饰
- **透明度**：3-5%（极淡，不可喧宾夺主）
- **生成方式**：SVG 矢量，非位图

### 8.2 水墨效果

- **类型**：极淡的水墨晕染
- **位置**：卡片背景、分隔线装饰
- **透明度**：2-4%
- **CSS 实现**：`filter: blur(80px)` + `opacity: 0.03`

### 8.3 金色装饰线

```
─── ✦ ───
```
- 用于章节分隔、卡片内装饰
- 金色 `#D4A853`，宽度 `1px`，两端小星号装饰

## 9. 动效规范

### 9.1 原则

- **快**：所有过渡 ≤ 300ms
- **缓动**：使用 `cubic-bezier(0.4, 0, 0.2, 1)`（Apple 标准）
- **有意义**：每个动画都有目的，不为动而动

### 9.2 关键动效

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面加载 | 内容从下方淡入（stagger） | 400ms |
| 卡片 Hover | 背景变亮 + 边框发光 | 200ms |
| 按钮 Click | scale(1→0.98→1) + 亮度变化 | 150ms |
| 输入框 Focus | 边框变色 + 光晕扩散 | 250ms |
| 结果加载 | 骨架屏 → 内容渐显 | 500ms |
| 付费引导 | 金色边框脉冲（循环 3 次） | 2s × 3 |
| 页面切换 | 内容滑入 + 淡入 | 300ms |

### 9.3 骨架屏（Skeleton）

```
┌──────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │  标题
│ ▓▓▓▓▓            │  副标题
│                  │
│ ┌────┐ ┌────┐    │  卡片骨架
│ └────┘ └────┘    │
└──────────────────┘
```

- **背景**：`#151520`
- **骨架**：`#1C1C2A` + `background-size: 200px 100%` + `animation: shimmer 1.5s infinite`
- **文字区域**：宽度递减的圆角矩形

## 11. 设计检查清单

每次设计评审时，确认以下要点：

- [ ] **暗色主题**是否符合色彩规范？（背景 `#0A0A0F`，文字 `#E8E4DF`）
- [ ] **金色强调**是否克制？（全页金色元素不超过 5 处）
- [ ] **留白**是否充足？（行间距 ≥ 1.6，区块间距 ≥ 24px）
- [ ] **圆角**是否统一？（卡片 16px，按钮 12px，输入框 12px）
- [ ] **毛玻璃**效果是否可用？（`backdrop-filter: blur(20px)`）
- [ ] **字体**是否正确？（英文 Inter，中文 Noto Sans SC）
- [ ] **五行配色**是否准确？（木绿/火红/土金/水蓝/金）
- [ ] **响应式**是否完整？（375px / 768px / 1280px 三档）
- [ ] **动效**是否克制？（≤ 300ms，缓动曲线一致）
- [ ] **无障碍**是否考虑？（对比度 ≥ 4.5:1，焦点可见）
