# Paper Summarizer

A free AI-powered research paper summarizer that helps researchers, students, and academics quickly extract key findings from academic papers.

## Features

- 📄 Upload PDF or paste text
- 🤖 AI-powered summaries powered by DeepSeek
- 🔍 Key findings, methodology, and conclusions extraction
- 🌐 Multi-language support (English & Chinese)
- ⚡ Fast processing (10-30 seconds)

## Try it live

Visit [summarizeai.app](https://www.summarizeai.app) for the full experience!

## Tech Stack

- Next.js 14
- DeepSeek API
- Supabase
- Tailwind CSS
